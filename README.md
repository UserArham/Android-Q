# This OS actually does not use the Scratch VM.

Instead, it uses a mod called Turbowarp. Turbowarp is a mod that optimized the Scratch VM by dropping use of RAM.


## Why I use TurboWarp?

In my older repo which is archived, I used the un-optimized Scratch 3.0 engine that is eating RAM and is also a resource hog. So today, I optimized the whole thing by using TurboWarp, a optimized version of Scratch 3.0.
